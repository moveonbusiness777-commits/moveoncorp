プライバシーポリシー

最終更新日：2025年11月25日

本プライバシーポリシーは、[数字パズル〜6×6・9×9レベルアップと可愛いイラスト〜]（以下「本アプリ」といいます）の利用に伴い、開発者 [MOVE ON株式会社]（以下「当方」といいます）が取得・利用する情報の取り扱いについて定めるものです。

本アプリをご利用になる前に、本プライバシーポリシーをよくお読みください。

1. 取得する情報

本アプリでは、以下の情報を取得・保存する場合があります。

1-1. ユーザーから直接ご提供いただく情報

本アプリは、パズルゲームの提供を主目的としており、通常のプレイにおいてユーザー登録や氏名・住所などの入力を求めることはありません。

ただし、サポートお問い合わせ時に、ユーザーが任意で以下の情報を提供する場合があります。

  • お名前またはニックネーム
  • メールアドレス
  • お問い合わせ内容
  • 不具合報告のためにユーザーが任意で記載した情報
    （ご利用端末名、OSバージョン、発生日時、画面のスクリーンショット など）

また、iOS の「Appleでサインイン」を利用した場合、Apple から提供されるメールアドレスなどの情報が Firebase アカウント情報として保持されることがありますが、当方がこれを広告配信や不要な連絡に利用することはありません。

1-2. アカウント・識別子に関する情報

本アプリでは、ゲームデータの保存・復元や複数端末での利用のために、以下の識別子を利用する場合があります。

  • Firebase Authentication によって発行されるユーザーID（匿名IDを含む）
  • アプリ内でランダムに生成される端末ID（例：`dev_xxxxx...`）
    ※氏名やメールアドレスとは紐付けず、ゲームデータのひも付けのみに利用します。
  • Appleでサインイン利用時に、Apple から Firebase へ提供される署名付きトークン等の情報

これらの識別子は、主に以下の目的で利用されます。

  • ゲーム進行状況・ポイント・設定のサーバー保存および復元
  • 複数の端末で同じゲームデータを利用できるようにするため
  • 不正利用や不具合調査のための技術的な確認

1-3. ゲームプレイ・進行状況に関する情報

本アプリでは、ゲーム提供のために次のような情報を端末内およびサーバー（Firebase Firestore 等）に保存します。

  • ゲーム進行状況
    • クリアしたステージ・章
    • 現在プレイ中のパズルID・インデックス
    • 獲得したポイント・経験値 など
  • ゲームプレイ統計（KPI）
    • クリアしたパズルの累計数
    • 正解・不正解の回数
    • プレイ時間
    • 初回プレイ日・最終プレイ日 など
  • ユーザー設定
    • 効果音・バイブレーションの ON/OFF
    • 文字サイズ
    • 入力モード
    • 「ゲーム改善のためのデータ共有」への同意状況
    • 「広告のカスタマイズ」への同意状況 など

これらは、ゲーム体験の提供・継続のために必須の情報であり、氏名・住所などの直接的な個人情報は含みません。

1-4. アプリ利用時に自動的に取得される情報

本アプリでは、品質向上や不具合解析、広告配信のために、以下の情報を自動的に取得する場合があります。

  • 端末情報
    • 端末モデル、OSバージョン、言語設定 等
  • アプリ情報
    • アプリのバージョン、実行環境（iOS/Android/Web 等）
  • アプリの利用状況
    • 画面遷移、プレイ回数、プレイ時間
    • クリア率、ヒント使用回数、ミス回数
    • デイリーチャレンジの利用状況 など
  • 課金・購入に関する情報
    • 購入したアイテムやサブスクリプションの有無
    • 広告削除などのエンタイトルメント状態
      （※決済自体は Apple / Google が行い、当方はクレジットカード番号等にはアクセスしません）
  • 広告に関する情報
    • 広告の表示回数・視聴完了有無（報酬広告など）
    • 広告ID（Advertising ID / IDFA 等：広告が有効な場合）

これらの情報は、個人を特定することを主目的とせず、統計的な分析やサービス改善のために利用されます。

2. 情報の利用目的

当方は、取得した情報を以下の目的で利用します。

  • 本アプリの提供・運営のため
  • ゲーム進行状況・ポイント・設定の保存および復元のため
  • 複数端末間でのデータ同期のため
  • 本アプリの品質改善・不具合対応・新機能開発のため
  • 利用状況の分析によるゲームバランス調整・ユーザー体験の向上のため
  • お問い合わせへの回答およびサポート対応のため
  • アプリ内課金の管理・購入状況に応じた機能解放（広告削除など）の判定のため
  • 広告の表示およびその効果測定のため（広告を導入している場合）
  • 法令に基づく場合その他これに準ずる正当な理由のため

なお、「ゲーム改善のためのデータ共有」や「広告のカスタマイズ」に関するデータ送信は、アプリ内の同意画面および設定画面から、ユーザーがいつでも ON/OFF を切り替えることができます。
オフにした場合でも、ゲームの基本的なプレイ自体は可能です。

3. 外部サービスの利用について

本アプリでは、以下の外部サービスを利用します。
各サービスが取得・利用する情報の範囲および取り扱いについては、各社のプライバシーポリシーをご確認ください。

3-1. Firebase（Authentication / Firestore / Analytics など）

提供者：Google LLC

利用目的：

  • 匿名ログインおよび「Appleでサインイン」によるユーザー認証
  • ゲーム進行状況・ポイント・設定等のクラウド保存および復元
  • アプリの利用状況の匿名的な分析
  • 不具合発生時の原因調査（クラッシュログ等を導入した場合）

Firebase のプライバシーポリシー：
https://policies.google.com/privacy

3-2. Google Mobile Ads（Google AdMob：広告配信）

提供者：Google LLC

利用目的：

  • 本アプリ内に表示する広告（バナー広告・報酬型広告など）の配信
  • 広告配信の最適化および効果測定

Google Mobile Ads / AdMob は、広告ID等を利用し、ユーザーの興味・関心に基づく広告（パーソナライズド広告）を表示する場合があります。

本アプリでは、初回起動時や設定画面において「広告のカスタマイズ」の可否を選択でき、オフにした場合は、パーソナライズされていない一般的な広告のみが表示されます。
また、iOS / Android の OS 設定から、トラッキングや広告IDのリセットも行えます。

Google 広告に関するポリシー：
https://policies.google.com/technologies/ads

3-3. RevenueCat（アプリ内課金管理）

提供者：RevenueCat, Inc.

利用目的：

  • アプリ内の購入履歴・有効なサブスクリプションの管理
  • 購入状況に応じた機能制限（広告削除など）の判定
  • 同一ユーザーによる複数端末での購入復元

RevenueCat を通じて、購入に関する情報（購入した商品IDや有効期限等）が当方に共有されますが、クレジットカード番号などの決済情報は当方では保持せず、Apple / Google / 各ストアの決済システム側でのみ取り扱われます。

RevenueCat のプライバシーポリシー：
https://www.revenuecat.com/privacy/

4. 個人情報の第三者提供

当方は、以下の場合を除き、取得した情報を第三者に提供することはありません。

  • ユーザー本人の同意がある場合
  • 法令に基づく場合
  • 人の生命、身体または財産の保護のために必要であり、本人の同意を得ることが困難である場合
  • 公衆衛生の向上または児童の健全な育成の推進のために特に必要であり、本人の同意を得ることが困難である場合
  • 国の機関もしくは地方公共団体またはその委託を受けた者が、法令の定める事務を遂行することに対し協力する必要がある場合

5. 情報の管理

当方は、取得した情報について、漏えい・滅失・毀損が行われないよう、必要かつ適切な安全管理措置を講じます。

必要がなくなった情報については、合理的な範囲で、適切な方法で削除または匿名化いたします。

ユーザーからご本人に関するデータの削除をご希望いただいた場合には、本人確認のために必要な情報を確認のうえ、可能な範囲で対応します。

6. 未成年者の利用について

本アプリは、幅広い年齢層を対象としたパズルゲームですが、未成年のユーザーが利用する場合には、保護者の方の同意を得た上でご利用いただくことを推奨しています。

また、課金機能・広告表示を含むため、保護者の方による端末の利用制限設定もご検討ください。

7. プライバシーポリシーの変更

当方は、必要に応じて本プライバシーポリシーを変更することがあります。
重要な変更がある場合は、本アプリ内または当方が運営するWebサイト等でお知らせします。

変更後のプライバシーポリシーは、当方が別途定める場合を除き、本ページに掲載されたときから効力を生じるものとします。
